# The-Prompt-Shelf

> Curated gallery of AI coding rules — CLAUDE.md, .cursorrules, AGENTS.md — from top open-source projects.

**[thepromptshelf.dev](https://thepromptshelf.dev)**

As AI coding assistants become essential developer tools, well-crafted rules and prompts make a real difference in code quality. But good examples are scattered across GitHub repos, blog posts, and forums. The-Prompt-Shelf collects them in one place.
